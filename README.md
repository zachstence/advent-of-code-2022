# advent-of-code-2022

My solutions to [Advent of Code 2022 problems](https://adventofcode.com/2022).

## Usage
```sh
# Run solution for current day
cargo aoc

# Run solution for specific day/part
cargo aoc -d 2 -p 1
```

More docs at https://github.com/gobanos/cargo-aoc
